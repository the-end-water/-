# 像素游戏《出埃及记》项目介绍

> - YT视频链接：<https://youtu.be/VntNX9FIn2k>
## 项目灵感
1. 一年多前曾经使用pygame进行五子棋的游戏制作，对于pygame游戏开发有了一些初步了解，所以一直好奇pygame能否进行一些类似于像素风格的rpg游戏制作。
2. 闲暇时间曾经看到使用pygame进行"星露谷物语"的游戏开发，认识到pygame具有比较完善的开发能力。
3. 此次犹太文明课程上面，了解了犹太民族的起源，对于先人摩西的经历充满了兴趣，而有关摩西最为知名的便是《出埃及记》，所以借着课程设计的机会，尝试着把这段故事使用rpg风格的游戏进行展示。

## 声明
- 本人非计算机专业或者游戏设计专业人员，在编程过程中设计到一些具体的算法，可能非常臃肿，不够简洁高效，本人已经尽力制作。
- 游戏运行逻辑上，本人基于最朴素的游戏逻辑制作，尤其是人物对话，场景切换，动画加载等方面，很多地方显得不够专业，敬请谅解。
- 游戏全程由本人单独完成，包括所有的音乐插入，地图设计，角色设计和贴图，交互逻辑等等，很多地方显得粗糙，敬请谅解。
- 作品仅仅关于《出埃及记》以及摩西有关经历做出展示，中间加入了适量改编使得剧情完整。全作品不涉及任何种族、民族、国家、政治看法和特殊立场。
## 游戏特点
1. 未使用任何的商业化的游戏引擎（例如RPGmaker，Unity，等等），仅仅基于pygame编写，体量小，运行快。
2. 添加了可随时触发的过场视频，增强故事的完整度。
3. 在地图创建上面，未使用任何现成的的地图素材，而是基于“埃及”“沙漠”等元素，在Tiled进行手动地图编制，地图场景更加贴合故事背景。
## 技术路线
> - 总共耗时约32小时
> - 基本展现了较为完整的故事
### 1. 文案设计
----
- 主要调研了《出埃及记》电影，民间解读版本，国外创意视频，综合形成的文案。
- 核心情节包括：
    - 摩西幼年时被埃及公主抚养长大
    - 成年后因为一次意外发现自己的身世
    - 为了保护以色列奴隶而杀害埃及士兵，被迫离开埃及
    - 流浪到西奈山，在山顶得到了上帝的启示
    - 重返埃及，团结以色列人，并且告诫法老
    - 法老拒绝摩西的告诫，受到十大灾害，最终同意以色列人离开
    - 法老最终背弃誓言，追杀以色列人到红海边
    - 摩西得到神意，开辟大海让以色列人通过，淹没埃及追兵
### 2. 地图设计
----
- 主要包含6张地图（存在地图复用）：
    - **埃及宫殿**
    - **埃及采石场**
    - **以色列人村落**
    - **埃及城市**
    - **西奈山**
    - **红海边(因为技术原因，非常粗糙)**
- 地图使用**Tiled**软件进行独立编辑创作，基础素材来源于民间资源库。
- 因为技术能力有限，地图的**碰撞箱设计**采取平面矩形碰撞检测。
### 3. 角色设计
----
- 核心人物摩西：因为没有找到合适的中年形象，所以全程采用一个青年形象，有所偏颇敬请谅解。
- 其他人物包含：公主，大皇子，亚伦，米利安，以色列族人，埃及士兵等等。
- 人物形象来自于素材网站，本人进行了动画帧处理。
- 对于摩西人物。基本实现了在静止以及移动时，角色拥有不同的动画状态，同时可以实现人物交互，碰撞检测。
### 4. 过场视频
----
- 在不同地图切换的时候，插入了进度条来作为转场。
- 同时在每个地图的剧情结束后，插入了**视频补充**，视频来源为电影 **《出埃及记》** 节选片段。
- 受限于技术能力，很多大型画面难以实现，因此依靠**视频补充**来进行描述。
- 视频在地图中有专门的触发区域。
### 5. 背景音乐选择
----
- 选择了电影 **《出埃及记》** 的主题曲《Ernest Gold：Theme Of Exodus》
- 帕特·布恩演唱，帕特·布恩填词，Ernest Gold谱曲
## 游戏基本说明
- 本游戏是完全基于行走与对话式触发的类rpg游戏，不包含战斗情节。
- 由于事件触发的检测范围没有找到最合适的区域，因此可能要贴紧npc才能触发对话。
