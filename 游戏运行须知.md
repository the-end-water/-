# 像素游戏《出埃及记》运行须知

## 依赖文件（python安装）
- python 需要3.10及以上版本

- 在python 3.10的基础上，安装：numpy, pygame, cv2, sys

- 你可以基于以下命令进行安装
```python
pip install numpy
pip install pygame
pip install opencv-python
pip install sys
```
## 正式游玩
- 打开项目文件里面的**main.py**文件，点击运行即可。（主义不要移动其他文件夹的文件，否则会造成路径识别问题）
- 游玩过程主要依靠**人物对话**，**过场视频**触发。
- 每个地图会提示和哪些npc进行对话，以及对话的顺序。（并非所有npc都可以对话）
- 靠近npc，按下空格即可触发对话。（有时候会出现检测范围的bug，贴近npc多按几下空格就好了）
- 结束每个地图的所有对话后，在地图中会有**棕色矩形（过场视频）** 和 **粉色矩形（传送点）** 。走到对应的矩形上面即可触发。（注意：先触发过场视频，再进行传送）
